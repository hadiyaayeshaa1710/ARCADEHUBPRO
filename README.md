# Arcade Hub Pro 🎮

Arcade Hub Pro is a modern arcade-style desktop gaming application developed using **C# Windows Forms**. The application combines multiple mini-games into a single interactive platform with a neon-themed graphical user interface, smooth animations, responsive controls, and real-time score tracking.

The project includes three integrated mini-games:

* 🔍 **Word Search** — Find hidden words inside a dynamically generated letter grid.
* 🧠 **Memory Match** — Flip cards and match identical pairs.
* 🎈 **Balloon Pop** — Pop rising balloons before they escape.

The application features animated backgrounds, glowing neon UI effects, timers, score systems, hover animations, and modular game architecture. It demonstrates concepts such as event-driven programming, GUI development, custom graphics rendering, timers, and object-oriented programming using Windows Forms.
